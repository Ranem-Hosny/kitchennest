# بيت العوضي — KitchenNest 🏠

متجر إلكتروني لأدوات ومستلزمات المطبخ (حلل، مقالي، أطقم مائدة، أدوات تقديم…) بواجهة عربية RTL، مع لوحة تحكم كاملة لإدارة المنتجات والطلبات.

## المكوّنات

- **الواجهة الأمامية** — صفحات HTML/CSS/JS ثابتة (`index.html`, `category.html`, `product.html`, `cart.html`, `checkout.html` …).
- **لوحة التحكم** — `admin/` (PHP): إدارة المنتجات، التصنيفات، العروض، **المجموعات المختارة**، الطلبات، والرسائل.
- **الـ API** — `php/` : حفظ الطلبات، رسائل التواصل، وتغذية الواجهة بالمنتجات والمجموعات.
- **قاعدة البيانات** — MySQL (`db/schema.sql` + `admin/setup.php`).

## التشغيل محلياً (XAMPP)

1. شغّل **Apache + MySQL** من XAMPP.
2. انسخ المشروع داخل `htdocs`، أو شغّله بسيرفر PHP المدمج:
   ```bash
   php -S localhost:8000 -t .
   ```
3. أنشئ قاعدة البيانات وجداولها بزيارة صفحة الإعداد مرة واحدة:
   ```
   http://localhost:8000/admin/setup.php
   ```
   تُنشأ الجداول ومستخدم افتراضي: **admin / admin123** (غيّره فوراً).
4. افتح الموقع: `http://localhost:8000/`
   واللوحة: `http://localhost:8000/admin/`

## الإعدادات

- بيانات الاتصال بقاعدة البيانات في `php/config.php` و `admin/includes/admin-config.php`
  (الافتراضي: `localhost` / `root` / بدون كلمة مرور — عدّلها قبل النشر الفعلي).
- رقم واتساب، InstaPay، ورسوم الشحن تُضبط من **إعدادات المتجر** داخل اللوحة.

## ملاحظات

- مجلد `uploads/payments/` (إثباتات دفع العملاء) **مستبعد** من المستودع للخصوصية.
- بيانات قاعدة البيانات (المنتجات/الطلبات) لا تُرفع مع الكود — أعِد إنشاءها عبر `setup.php` أو استورد نسخة SQL.
